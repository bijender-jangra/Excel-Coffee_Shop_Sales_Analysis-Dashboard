# Coffee_Shop_Sales_Analysis
☕ Interactive Coffee Sales Analysis Dashboard built in Microsoft Excel using Power Query, Pivot Tables, Pivot Charts, and Slicers. Features KPI tracking, sales trends, product performance, location-wise analysis, hourly sales insights, and dynamic filtering for business intelligence. 📊🚀
